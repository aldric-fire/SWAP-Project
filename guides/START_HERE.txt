========================================
‚ö° QUICK SETUP (5 MINUTES)
========================================

STEP 1: Start XAMPP
-------------------
1. Open XAMPP Control Panel
2. Click "Start" next to Apache
3. Click "Start" next to MySQL
4. Both should show green "Running" status

STEP 2: Copy Project
--------------------
Extract and copy the entire folder to your XAMPP htdocs directory:
- Windows: C:\xampp\htdocs\
- Mac: /Applications/XAMPP/htdocs/
- Linux: /opt/lampp/htdocs/

STEP 3: Create Database
-----------------------
1. Open browser: http://localhost/phpmyadmin
2. Click "SQL" tab
3. Open database.sql file from project folder
4. Copy ALL the SQL code
5. Paste into phpMyAdmin SQL box
6. Click "Go" button
7. Verify "products_db" appears in left sidebar

STEP 4: Configure Base URL
---------------------------
IMPORTANT: Update the BASE_URL to match your folder name!

1. Open: config/config.php
2. Find: $base = '/sample_project';
3. Change to match YOUR folder name, if you renamed it
4. Save the file

STEP 5: Open the Website
-------------------------
1. Open browser
2. Go to: http://localhost/your-folder-name/
   Example: http://localhost/sample_project/
3. You should see 5 sample products!

üîß TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem: CSS not loading
Solution:
- Open config/config.php
- Verify BASE_URL matches your folder name
- Clear browser cache (Ctrl+Shift+R)

Problem: Database connection failed
Solution:
- Check MySQL is running in XAMPP
- Verify credentials in config/database.php
- Make sure you ran database.sql

Problem: Page is blank
Solution:
- Check PHP error logs
- Look for syntax errors (missing semicolons)
- Enable error display in PHP

Problem: Form doesn't submit
Solution:
- Check form method="POST"
- Verify input names match $_POST keys
- Check browser console for JavaScript errors
